# Guess Number Game

## install the game

Clone the repository

```
git clone https://github.com/Baddou37/guessnumber.git
```

move to guessnumber folder

```
cd guessnumber
```

install dependencies

```
npm install
```

## Play the game

start the game with node.js and index.js file

```
node index.js
```

Have fun !
